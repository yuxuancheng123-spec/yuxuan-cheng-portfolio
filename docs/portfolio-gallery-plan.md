# Portfolio Gallery Home Plan

## Goal
Create a standalone personal portfolio homepage for Yuxuan Cheng with an animated project gallery, personal introduction, and smooth project-detail reveal.

## Scope
- Static website using `index.html`, `styles.css`, and `app.js`.
- No build step or external package required.
- The first featured project links to the existing AI Actor Compliance Workspace.
- Project cards show preview thumbnails, skill tags, and hover motion.
- Clicking a card opens an in-page detail panel instead of navigating away immediately.

## Initial Sections
- Hero: name, role positioning, and short introduction.
- Project gallery: animated thumbnail cards.
- Detail drawer: selected project overview, skills, deliverables, and links.
- About strip: concise profile and focus areas.
- Contact/footer: GitHub and email placeholders.

## Visual Direction
- Gallery-first layout inspired by polished creative portfolios.
- Calm, professional, monochrome base with one blue accent.
- Hover previews should feel tactile and smooth.
- No dependency on proprietary brand styling.

## Acceptance Checks
- Page opens from a local file or simple local server.
- Hover states work on project cards.
- Clicking a project opens the detail panel.
- AI Actor Compliance card links to the existing demo path.
- Mobile layout has no horizontal overflow.
