# Portfolio Gallery Home Plan

## Goal
Create a standalone personal portfolio homepage for Kenny Cheng with a centered floating-widget layout, resume entry point, and smooth panel switching.

## Scope
- Static website using `index.html`, `styles.css`, and `app.js`.
- No build step or external package required.
- The portfolio widget links to the existing AI Actor Compliance Workspace.
- The resume widget links to `assets/Kenny_Cheng_Resume.pdf`.
- The project stack can scatter and collect project shortcuts.
- Sidebar navigation switches content panels without a page reload.

## Initial Sections
- Left navigation widget.
- Center greeting widget.
- AI identity visual card.
- Clock and calendar widgets.
- Latest work, portfolio, resume, and contact widgets.

## Visual Direction
- Center-cluster layout inspired by soft personal desktop dashboards.
- Calm mint, aqua, white, and glass-like surfaces.
- Hover states should feel tactile and smooth.
- No dependency on proprietary brand styling.

## Acceptance Checks
- Page opens from a local file or simple local server.
- Hover states work on floating widgets.
- Sidebar panel switching works.
- Project stack scatter/collect works.
- AI Actor Compliance card links to the existing demo path.
- Mobile layout has no horizontal overflow.
